What is Text?

Text is a sequence of characters used to store words, symbols, or sentences.

Example:"hello world"

•Index•

UNI uses a character indexing system to locate text positions

Every character inside a text has its own position number.

Example:Hello World

Result:1 = H,2 = e,3 = l,4 = l,5 = o,6 = [space],7 = W,8 = o,9 = r,10 = l,11 = d]

start's with 1 and required a space

Expression → Text → Join

Join use to Join 2 or more different text

Example:Join(hello),(world)

result:helloworld

Expression → Text → Get Date

Get date use for Geting current date on your OS

Example:GetDate(YYYY/MM/DD)

Note: YYYY=Year, MM=Month, DD=Day

result:2026/1/28(example)

Expression → Text → Get Date Time

Get date times using to get current Date times like Hour,Minutes,Seconds on your OS

Example:Get Date Times(HH:MM:SS)

Note: HH=Hours, MM=Minutes, SS=Seconds

result:20:18:55(example)

What is number?

UNI uses number for mathematical concept and value used to count, measure, and label quantities

Example:10

Disclaimer: Decimal separator may depend on your device language or region settings

Some devices use "." while others use "," for decimal numbers

Expression → Number → mathematical operations

mathematical operation use for modify the value.mathematical operation like summation/subtraction/multiplication/divide

Example:10+10

result=20

Expression → Number → pick random

using for pick random number between selected number

Example:1 to 100 pick random integer

result=88

Expression → Number → remainder

Returns the remainder after dividing one number by another.

Example1:10/4

result=2 .because 10/4=2 remainder 2

Example2: 10/2

result=0 .because 10/2 = 5 remainder 0

Note:Similar to % (modulo) in many programming languages such as Lua, Luau, JavaScript, and Python.

Expression → Number → Sin/Cos

Sin

The Sin block returns a value between -1 and 1 based on an angle. Because the value changes smoothly over time.

Examples&Result:Sin(0) = 0 ,Sin(90) = 1 ,sin(180) = 0 ,Sin(270) = -1

Cos

The Cos block returns a value between -1 and 1 based on an angle. It works similarly to Sin, but starts at a different position in the wave cycle.

Examples&Result:cos(0) = 1 , cos(90) = 0 , cos(180) = -1 , cos(270) = 0

Sin Cos Graph

Note:Sin(blue),Cos(Red)

Expression → Number → PI

Returns the mathematical constant Pi (π)

Example:PI = 3.14159265...

Expression → Number → touch (X/Y) Position

Touch X Position

Returns the current horizontal touch position on the player's screen.

Touch Y Position

Returns the current vertical touch position on the player's screen.

Usage

Touch X Position and Touch Y Position can be used as an alternative way to get the player's touch position without using a Touch Event. this is useful when Touch Events are unavailable or when the player is interacting with UI elements that prevent Touch Events from firing.

Notes:Requires continuous updates (looping) to track movement. Can be accessed from different scripts. Works while interacting with UI elements. Can be used without Touch Events.

Logic Operators

And

If one condition is False, the result will be False.

Example:

if true and true = true

if true and false = false

if false and false = false

Or

If one condition is True, the result will be True.

Example:

if true or true = true

if true or false = true

if false or false = false

note:"or"&"and" can also be stacked . Example: if true and (true or true)

Returns a reference to a 3D world object or a 2D UI object. This expression is commonly used to access object properties, and components.

It can be used to interact with existing objects in the world or retrieve components attached to an object.

Returns the system object, which provides access to built-in engine features and APIs.

Properties

System properties is a lighting system and enviroment

BGM (Background Music)

Gets or changes the current background music of the map.

BGM Volume

Controls the volume of the background music.

Range: 0 - 100

Note: The maximum volume is 100.

Sky

Changes the Skybox displayed around the map.

Note: The Skybox is a visual background and is separate from Distant View.

Sky Fog Intense

Controls the fog visibility applied to the Skybox.

Note: This property only affects the Skybox.

Info: Horizon Height is not scriptable in Uni 2.53.

Please submit a report if Horizon Height becomes scriptable in future versions.

Light Color

Changes the color of the main directional light.

This affects the overall lighting color of the scene.

Light Intensity

Controls the brightness of the main light source.

Higher values make the map brighter.

Light Angle

Controls the vertical position of the light source.

This can be used to simulate sunrise, sunset, or midday lighting.

Light Direction

Controls the horizontal rotation of the light source.

Changes the direction from which shadows and light are cast.

Ambient Color

Changes the ambient light color of the scene.

Ambient light affects areas that are not directly lit by the main light.

Note: Darker ambient colors may appear more transparent.

Distant View

Displays a large decorative 3D background beyond the playable area.

Available types:

  • None
  • City/Hills
  • Space
  • Desert

Note: Distant View is look like a 3D object and not part of the Skybox.

Note: Distant View does not scale with map size. On very large maps, players may be able to get close to it.

View Direction

Rotates the Distant View horizontally.

Range: 0° - 360°

Filter

Applies a visual filter to the rendered scene.

Available modes:

  • Normal
  • Pixelated

Note: Filters affect Distant View.

Note: Filters do not affect the Skybox.

Fun Fact: Pixelated mode may improve performance on some devices.

Fun Fact: Pixelated mode adds pixel-style outlines to 3D objects also distant view.

Floor Size

Controls the size of the base floor.

Available sizes:

  • Hide
  • Small
  • Medium
  • Large

Note: Floor Size uses presets and cannot be set using Vector3 values.

Floor Color

Changes the color of the floor.

Tint Mode

Controls how floor coloring is applied.

Available modes:

  • Overlay
  • Tint

Floor Texture

Changes the texture used by the floor.

Weather

Controls the current weather effect of the map.

Available types:

  • None
  • Drizzle
  • Rain
  • Snow

Fog Color

Changes the color of the fog.

Objects inside the fog will gradually blend with this color.

Fog Distance

Controls how far the fog starts from the camera.

Higher values make nearby objects disappear into the fog sooner.

Fog Intense

Controls the visibility and transparency of the fog.

Higher values make the fog stronger and easier to notice.

Fog Bias

Controls the thickness and density of the fog.

Higher values make the fog appear heavier.

Note: Fog effects also affect Distant View objects.
